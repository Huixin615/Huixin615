# GitHub Profile Setup

This directory is ready to publish as the special GitHub profile repository:

```text
Huixin615/Huixin615
```

## Publish Steps

1. Create a public GitHub repository named `Huixin615`.
2. Push this directory as the repository contents.
3. Open the repository Actions tab and run `Generate contribution snake` once.
4. The profile README will appear on `https://github.com/Huixin615`.

## Notes

- The contribution snake image appears after the workflow creates the `output` branch.
- The stats cards and typing banner are remote SVG services, so GitHub renders them directly in the README.
